# Éclat Studio

Site vitrine moderne pour un institut de beauté et bien-être, construit en HTML, CSS et JavaScript vanilla.

## Fonctionnalités
- Responsive design mobile-first
- Mode clair/sombre avec persistance localStorage
- Système multilingue FR/EN
- Formulaire de contact et réservation fonctionnels côté client
- Structure modulaire et accessible

## Structure
- index.html : page d’accueil
- pages/ : pages de contenu et formulaires
- css/ : styles principaux, responsive et thèmes
- js/ : logique d’internationalisation, thème et formulaires
- assets/images/ : visuels du projet

## Déploiement
Le site peut être publié sur GitHub Pages, Vercel ou Netlify en servant le dossier racine du projet.
